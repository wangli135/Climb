# 使用markdown写ppt  
逼格满满  

## 准备条件  
- 下载代码[sliders](https://github.com/teddy-ma/sliders.git)
- 安装node环境  

## 操作流程  
1. 执行npm install确保环境  
2. 进入src目录，创建markdown文件  
3. 执行npm run build创建文件  
4. 执行npm run previes开启本地服务器  
5. 打开文件，6666
